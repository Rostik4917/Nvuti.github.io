Начнем с загрузки базы данных

1. Заливаем файл  с базой  в phpmyadmin

Подключаем скрипт к базе

2. Заходим в /inc/bd.php

Настраиваем подключение к базе, теперь переходим на свой сайт

Регистрируем себе аккаунт и переходим обратно в phpmyadmin

Ищем свой аккаунт в таблице kot_user и выдаем себе админа (столбик admin должен равняться 1 у того юзера, который имеет доступ к админке)

Теперь настраиваем весь сайт через админку.. 


Переходим к настройке платежной системы - FreeKassa

Секретные ключи должны быть одинаковыми, это обязательное условие

URL для оповещений - http://ваш-сайт.ru/inc/fkassa.php

URL при успешной оплате http://ваш-сайт.ru/sucpay.php

URL при неуспешной оплате http://ваш-сайт.ru/danpay.php

Готово, если вы все сделали строго по пунктам - ваш сайт должен работать

Если же сайт не работает, то дело в хостинге, либо же в том, что вы допустили ошибку, выполняя какой-либо из пунктов
